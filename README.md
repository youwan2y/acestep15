# Ace-Step 1.5 - AI Music Generation Landing Page

A modern, SEO-optimized landing page for Ace-Step 1.5 music generation model.

## 🚀 Features

- 🎨 Dark theme with neon/gradient effects
- 🎵 Music waveform animations
- ⚡ Particle background effects
- 📱 Fully responsive design
- 🔍 SEO optimized with meta tags and structured data
- 🚀 Built with Next.js 14 and Framer Motion

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 🚀 Deployment

This project is configured for Vercel deployment. Simply push to your repository and import it to Vercel.
